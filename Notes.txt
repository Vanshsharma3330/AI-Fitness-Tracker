JAVASCRIPT v/s TYPESCRIPT:
Typescript is statically typed superset of Javascript. Browsers can't directly execute typescript, the code first must be transpiled into standard Javascript.
Javascript is dynamically types and errors are caught during application runtime whereas in Typescript errors are caught during compilation/coding.
